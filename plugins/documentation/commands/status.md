---
description: Show Relay docs corpus freshness and source status.
---

Call the `documentation_status` MCP tool and summarize the source, page count, cache age, and next action if any. Mention that persistent caching is off unless `RELAY_DOCUMENTATION_MCP_WRITE_CACHE=1` is set. Do not describe Relay account, workspace, or Comms status; this plugin only reads public docs.
